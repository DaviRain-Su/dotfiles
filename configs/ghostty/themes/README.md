# Ghostty 主题

## 内置主题

Ghostty 自带大量主题，使用命令查看所有可用主题：

```bash
ghostty +list-themes
```

## 推荐主题

### 暗色主题
- `Tokyo Night` - 深蓝色调，护眼
- `Dracula` - 紫色调，高对比度
- `Catppuccin Macchiato` - 柔和的暗色主题
- `GruvboxDark` - 复古风格暗色
- `One Dark` - Atom 编辑器风格

### 亮色主题
- `Catppuccin Latte` - 柔和的亮色主题
- `GruvboxLight` - 复古风格亮色
- `Solarized Light` - 经典亮色

### 特色主题
- `12-bit Rainbow` - 多彩渐变
- `CLRS` - 简洁清晰
- `Cyberpunk` - 赛博朋克风格

## 自定义主题

将自定义主题文件放在此目录，然后在主配置中引用：

```
theme = my-custom-theme
```

主题文件格式参考: https://ghostty.org/docs/config/reference#theme
